# nanoc
Learning to write a compiler by writing one for the c64

## Vice
```
	apt install vice

	apt install subversion
	git clone https://github.com/lharsfalvi/vice-roms-deb.git
	cd vice-roms
	./makedeb.sh
	dpkg -i ivce-roms*.deb

	x64
```
## Toolchain
`apt install cc65`

Demo in `examples/fl`
```
	make
	make run
```
